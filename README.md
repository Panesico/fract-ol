## About

> A program that lets you visualize 3 distinct fractals with zoom.

## Usage

```
make
./fractol <name of fractal> (In case of Julia)<coordinate x> <coordinate y>
```

## Contents

Fractals: Mandelbrot, Julia, Tricorn
